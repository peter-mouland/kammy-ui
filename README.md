# Kammy-UI

 > Components for the Kammy App
 
## Dev

### WebApp
 * `yarn mongo`
 * `yarn watch`
 * `yarn start:webapp:dev`

### StoryBook
 * `yarn start`

## Prod

### WebApp
 * `yarn build`
 * `yarn start:webapp:prod`
